系统的硬件组成

  1. 总线 bus

    贯穿整个系统的一组电子管道，称作总线，它携带信息字节并负责向各个部件间传递。总线可以以它的工作对象抽象地大概分成三类（实际上会复杂得多）：

    1. 系统总线
    2. 内存总线
    3. I/O总线

  2. I/O设备 I/O System

    I/O（输入/输出）设备是系统与外部世界的联系通道（键盘、鼠标、显示器、磁盘、网络适配器等等），每个I/O设备都通过一个控制器或适配器与I/O总线相连

  3. 主存 Memory System

    主存是一个临时存储设备，在处理器执行程序时，用来存放程序和程序处理的数据。可以从物理和逻辑两个角度去理解它。

    物理上来说，主存是由一组动态随机存储器(DRAM)芯片组成的。
