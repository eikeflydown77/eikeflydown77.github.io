计算机组成与系统概述

概述

计算机系统是由硬件和软件构成的，他们共同工作来运行应用程序，虽然系统的具体实现方式随着时间而不断改变，但是系统内在的概念却没有改变，所有计算机系统都具有相似的硬件和软件组建，他们又执行着相似的功能。

计算机组成

  1. 总线 bus

    贯穿整个系统的一组电子管道，称作总线，它携带信息字节并负责向各个部件间传递。总线可以以它的工作对象抽象地大概分成三类（实际上会复杂得多）：

    1. 系统总线
    2. 内存总线
    3. I/O总线

  2. I/O设备 I/O System

    I/O（输入/输出）设备是系统与外部世界的联系通道（键盘、鼠标、显示器、磁盘、网络适配器等等），每个I/O设备都通过一个控制器或适配器与I/O总线相连

  3. 主存 Memory System

    主存是一个临时存储设备，在处理器执行程序时，用来存放程序和程序处理的数据。可以从物理和逻辑两个角度去理解它。
    物理上来说，主存是由一组动态随机存储器(DRAM)芯片组成的。 逻辑上来说，主存就是一个线性的数组，每个字节都有其唯一的地址（数组索引）。

  4. 处理器 Central Processing Unit

    中央处理器，简称处理器(CPU)，是解释和执行存储在主存中指令的引擎。处理的核心是一个大小为一个字的存储设备(或寄存器)，称为程序计数器(PC)。在任何的时刻，PC都指向主存中的某条机器语言指令(即含有该条指令的地址)。

操作系统管理硬件

硬件通常依靠于操作系统提供的服务，所有应用程序对硬件的操作尝试都需要操作系统，操作系统有两个基本功能：

  1. 防止硬件被失控的硬件程序滥用 -- 异常控制
  2. 向应用程序提供简单一致的机制来控制复杂而又通常大不相同的低级硬件设备

操作系统通过几个基本的抽象概念(进程、虚拟内存、文件)来实现这两个基本的功能，如图，文件是对I/O设备的抽象表示，虚拟内存是对主存和磁盘I/O设备的抽象表示，进程则是对处理器、主存和I/O设备的抽象表示。
